# FPGA
## 题目目的
运用AD/DA技术，产生雷达信号进行DA输出，并利用AD进行采样，完成非相参积累运算和CFAR检测。

## 题目要求
1. 设计雷达PRI和波门时序
2. DA波形输出
3. AD控制采集
4. 积累与检测
5. 首先MATLAB和modelsim仿真成功后，加载到FPGA，观察结果。

## 开题报告
1. 打算实现的功能
2. 怎么实现？
    - 框图
    - 状态机
    - 输入输出：
        - 板上按键模拟输入/状态输入；
        - Signal Tap 输出波形/数码管，led显示状态（旋转表示运行状态）
        - 各个功能逐渐实现（基本-->其他）
3. 能否按时完成
4. 任务分工
5. 至少10页
